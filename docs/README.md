## Prerequisites

- Docker Desktop or Docker Engine running
- Rust toolchain 1.70 or later
- Dioxus CLI (`dx`)

## Quick Start

````bash
# Clone the repository
git clone https://github.com/MH0386/doctainr.git
cd doctainr

# Run the application
dx serve --platform desktop
````
