## About Doctainr

Doctainr is a native desktop application for managing Docker containers, images, and volumes. Built with Rust for performance and Dioxus 0.7 for modern UI, it provides a lightweight alternative to Docker Desktop.

### Key Features

- Real-time Docker resource monitoring
- Container lifecycle management (start, stop, view)
- Image and volume browsing
- Native desktop performance
- Minimal resource footprint
